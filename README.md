## âš™ï¸ 2. How The Website Works

The workflow of the website is simple:

1. A user signs up or logs in using Google Authentication.
2. The user can:

   * Report a lost item
   * Report a found item
3. The user uploads item details along with an image.
4. The image is stored in cloud storage (Cloudinary).
5. Item details are stored in the database (Firebase / Supabase).
6. Other users can browse or search items.
7. If someone finds a match, they can contact the owner.

The system is designed to be responsive and works on both desktop and mobile devices.
